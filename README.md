# Calculadora CLI Interactiva

Calculadora de consola construida en Python. Permite realizar operaciones 
básicas de forma interactiva hasta que el usuario decida salir.

## Funcionalidades
- Suma, resta, multiplicación y división
- Validación de entrada de datos
- Manejo de división por cero
- Interfaz de menú en consola

## Cómo ejecutar
```bash
python calculadora.py
```

## Tecnologías
- Python 3.12
- Manejo de errores con try/except
